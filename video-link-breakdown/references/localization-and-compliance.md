# 跨境本地化与合规

## 本地化顺序

先确定目标国家、语言、平台、品类、价格带和人群，不默认一次分析多个地区。需要比较市场时使用同一评价维度。

检查语言与语气、演员可信度、使用场景、音乐与声音、卖点顺序、价格和促销表达、社会证明、CTA 与购买路径。所有市场画像都应视为研究假设，并用当前市场、品类和账号数据校准。

## 本地化层级

- `L1 语言适配`：字幕或配音，保留原画面；适用于文化依赖低的通用演示。
- `L2 文化适配`：调整卖点、价格、音乐、文字、CTA 和重新剪辑；适用于购买动机有差异的市场。
- `L3 全面重制`：本地演员、场景和脚本；仅在信任、合规或文化差异要求时使用。

给出元素级改动、成本区间、币种、来源、周期和不可调整的硬障碍。没有当地报价时只输出成本等级和询价清单。

| 欧盟 | CE 标志、GPSR（通用产品安全法规 2024/1245）、REACH（化学品）、RoHS（电子有害物） | UCPD（不公平商业行为指令）、各国 ASA 自律体系（如英国 ASA） | GDPR | 在线平台责任（DSA）、消费者权利指令 |

| 日本 | PSE（电子）、食品卫生法、药机法（化妆品/医药品） | 景品表示法（不当表示）、药机法宣称限制 | APPI（个人信息保护法） | 特定商取引法（电商披露） |

法规引用为快筛起点，不是法律意见。高风险品类（食品入口、化妆品、医疗器械、儿童产品、电子产品）在进入市场前必须完成专业合规审查。

对于每个目标市场，回答：

1. 该视频的核心打法在该市场是否成立？
2. 需要调整哪些元素？（语言、演员、场景、音乐、卖点顺序、价格锚定）
3. 调整的预估成本和周期。
4. 是否存在不可调整的硬障碍？（文化禁忌、合规红线）

所有市场画像结论标注为 `I`（推断），除非有当前市场数据支撑。

## 合规快筛

产品层检查准入、认证/注册、标签、营销宣称、专利、商标、版权和平台禁限售。内容层检查广告关系披露、功效或比较宣称、儿童/医疗/金融等敏感内容、音乐和素材授权、文化与宗教风险。

当前法规、平台规则和披露要求必须联网核验并记录日期。Skill 只作商业快筛；高风险问题标记“合规否决”或“进入前复核”，并指定律师、检测机构、平台合规或本地化团队。

不提供隐藏广告关系、绕过审核、误导宣称或侵权搬运方案。

## 本地化效果验证

完成本地化适配后，需要验证调整是否有效。以下是验证框架：

### 上线前验证

| 检查项 | 方法 | 通过标准 |
|---|---|---|
| 语言自然度 | 母语者审校字幕/配音/文案 | 无语法错误、无直译痕迹、口语化自然 |
| 文化适当性 | 本地团队或文化顾问审查 | 无文化冒犯、无不当符号/颜色/手势 |
| 合规通过 | 对照目标市场法规清单逐项检查 | 无硬性违规项 |
| 产品-市场匹配 | 检查卖点是否匹配当地购买动机 | 核心卖点与当地用户关注点一致 |
| 技术合规 | 检查字幕格式、画面比例、音频标准 | 符合平台技术规格 |

### 上线后验证

| 指标 | 对比基准 | 行动阈值 |
|---|---|---|
| 3 秒播放率 | 与同市场同类目本地内容对比 | 低于基准 30% 以上需优化 Hook |
| 完播率 | 与同市场同类目本地内容对比 | 低于基准 20% 以上需优化节奏 |
| 互动率 | 与同市场同类目本地内容对比 | 低于基准 40% 以上需重新评估内容策略 |
| 转化率 | 与原语言版本或市场均值对比 | 低于基准 25% 以上需检查电商衔接 |
| 负面评论比例 | 是否出现文化不适/理解困难的反馈 | 出现即需排查具体元素 |

### 验证流程

1. **小批量测试**：先在目标市场投放少量本地化素材，观察核心指标。
2. **A/B 对比**：与原语言版本或不同本地化版本对比（如有条件）。
3. **定性反馈**：收集评论区反馈、本地团队/合作伙伴意见。
4. **迭代调整**：根据数据反馈调整本地化元素，而非直接放弃或全盘重做。
5. **记录结论**：将验证结果记录为 `B`（外部基准），供后续分析复用。

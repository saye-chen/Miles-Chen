# 视频商业决策

## 商业门槛

在讨论放量前判断：

1. 利润：完整成本后贡献利润是否为正。
2. 合规/IP：产品和内容宣称是否存在否决项。
3. 供应链：起量后的交期、库存和质检是否可承受。
4. 退货：视频承诺是否可能放大预期落差与退款。
5. 产品-视频匹配：格式、价格带、人群、卖点和差异是否匹配。

门槛不通过时，不因内容评分高而建议放量。

## 产品-视频匹配评估

内容质量高不等于商业价值高。当视频用于带货或推广特定产品时，按以下五个维度评估匹配度（1-10 分）：

| 匹配维度 | 判断问题 | 评分锚点 |
|---|---|---|
| 品类适配 | 视频的内容格式是否适合展示该产品？ | 1-3: 格式与品类严重错配（如标品用剧情号）；5-7: 基本适配但无亮点；8-10: 格式天然适合该品类（演示型产品用前后对比，体验型用真人使用） |
| 价格带适配 | 视频建立的信任深度是否匹配产品价格？ | 1-3: 信任深度严重不足（如 $100+ 产品只靠视觉冲击）；5-7: 基本匹配；8-10: 信任建设充分且有余（完整证据链、社会证明、风险逆转） |
| 人群适配 | 视频吸引的受众是否与产品目标人群重合？ | 1-3: 受众严重错位；5-7: 部分重合；8-10: 高度重合 |
| 卖点覆盖 | 视频是否覆盖了产品的核心卖点？ | 1-3: 核心卖点未覆盖；5-7: 覆盖了主要卖点；8-10: 覆盖完整且有差异化表达 |
| 竞争差异 | 该视频打法能否在同类产品中形成差异化？ | 1-3: 完全同质化；5-7: 有一定差异；8-10: 打法本身构成竞争壁垒 |

产品-视频匹配结论必须标注证据编号（`O/M/U/B/I`），不得把推断写成观察。

## 电商衔接审计

很多视频在内容漏斗上表现优秀（高完播、高互动），但在电商漏斗上断裂——用户看完觉得"好看"但没有"想买"。当视频用于带货时，必须审计内容漏斗与电商漏斗的衔接层：

| 检查项 | 判断要点 | 断裂后果 |
|---|---|---|
| 产品出现时间占比 | 产品在视频中的可见时长占比是否足够建立购买意向？ | 仅与该账号、品类和内容格式的成功/失败基线比较；20% 只能作为待验证示例，不得作为跨场景固定门槛 |
| 卖点传达清晰度 | 是否传达了一个明确的购买理由？不是"提到了产品"，而是"建立了购买理由" | 用户看完不知道为什么要买 |
| 价格锚定 | 用户看完视频后是否知道价格区间？是否建立了"值这个价"的认知？ | 点击后价格落差导致跳出 |
| CTA 到商品的路径 | 从"想买"到"能买"之间有几步？挂车位置、评论区链接、主页入口是否顺畅？ | 路径越长，转化损耗越大 |
| 承诺与体验一致性 | 视频承诺是否与产品实际体验一致？ | 不一致 → 高退货率 → 放量等于放大亏损 |

若衔接层断裂，在结论中明确指出"内容漏斗有效但电商衔接断裂"，并给出修复建议。不因内容评分高而跳过衔接审计。

## 电商漏斗

分析曝光 → 播放 → 有效观看 → 商品点击 → 加购 → 下单 → 签收 → 不退货。前三至四个环节可能受视频直接影响；Listing、价格、优惠、物流和评论基础需单列为视频外因素。

检查产品出现时机、证明强度、卖点清晰度、价格锚定、CTA 路径和承诺一致性。不要使用固定产品露出比例作为通用门槛；用同品类有效素材或账号基线校准。

## 单位经济

有输入时计算：

- 单件贡献利润 = 售价 - 产品 - 包装 - 物流 - 佣金 - 支付 - 退货准备金及其他变动成本；
- 可承受最高 CPA = 单件贡献利润 - 售价 × 目标净利润率；
- 盈亏平衡广告费率 = 单件贡献利润 ÷ 售价；
- 批次盈亏平衡订单 = 固定素材、达人和投流成本 ÷ 单件净贡献。

优先复用 CIDM 的 `profit_model.py` 与 `reverse_funnel.py`。缺少成本或漏斗输入时输出反推门槛，不用行业均值伪造精确利润。

## ROI

分别报告收入、贡献利润、净利、ROI 和净利润率。声明 ROI 分子采用归因收入还是归因贡献利润：

`ROI =（归因收入或归因贡献利润 - 内容与推广投入）÷ 内容与推广投入`

`净利润率 = 净利 ÷ 收入`

压力、基准、乐观场景必须明确曝光、CTR、CVR、签收、退货、单位贡献和投入来自 `M/U/B/I` 中哪一类。输入不足时只给敏感性或反推门槛。

## 结论

- `Go`：门槛通过且单位经济有安全垫，进入受控测试或放量。
- `Iterate`：内容或商业链存在可修复断点，给出一次或有限次数迭代。
- `Stop`：存在合规否决、负贡献、不可控退货或无法修复的产品-视频错配。

每个结论包含预算、指标、样本、复盘时间和停止条件。

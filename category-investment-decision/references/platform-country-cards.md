## 使用原则

本文件为高频使用的平台×国家组合提供专属卡片，包含该组合特有的费率、合规路径、竞争特征和证据优先级。当用户指定某个组合时，直接读取对应卡片，不必从 platform-playbooks.md 和 country-localization.md 分别提取再拼接。

**与现有文件的关系**：

- platform-playbooks.md 提供平台级经营逻辑（跨国家通用）。
- country-localization.md 提供国家级合规和本地化框架（跨平台通用）。
- 本文件提供**特定组合的专属信息**——只在该组合下成立的费率、合规路径、竞争特征。
- 遇到未覆盖的组合，分别读取 platform-playbooks.md 和 country-localization.md 拼接。

## 组合卡片结构

每张卡片包含：

| 字段 | 内容 |
|---|---|
| 组合标识 | 平台 + 国家（如 Amazon US、TikTok Shop US） |
| 费率结构 | 该组合特有的佣金、广告、物流、税费 |
| 合规路径 | 该组合特有的认证、标签、法规要求 |
| 竞争特征 | 该组合的头部集中度、新品突破难度、广告压力 |
| 证据优先级 | 该组合最重要的证据类型 |
| 卖家工具数据源 | 该组合可用的第三方数据源 |
| 常见失败模式 | 该组合特有的失败原因 |

### Amazon US

| 字段 | 内容 |
|---|---|
| **费率结构** | 佣金 8-15%（品类而异）；FBA 费按尺寸重量（标准尺寸 $3-8/件）；CPC 广告 $0.8-2.5；仓储费月 $0.56-2.40/立方英尺 |
| **合规路径** | FDA（食品/化妆品/医疗）、CPSC（儿童/消费品）、FCC（电子）、FTC（宣称）、EPA（杀虫/抗菌）；Prop 65（CA）；产品责任保险建议 |
| **竞争特征** | 头部集中度高（前 10 占 60-80% 销售）；Review 壁垒强（首页中位数 500-2000）；新品突破需 3-6 个月+广告投入 |
| **证据优先级** | 关键词簇搜索量、首页 Review 结构、新品突破案例、CPC 实际报价、FBA 费率 |
| **卖家工具数据源** | Helium 10、Jungle Scout、Keepa、Sellerboard、Brand Analytics |
| **常见失败模式** | 泛词硬打广告吞噬利润、Review 积累慢于预期、质量/退货失控导致账号风险、FBA 仓储费超预期 |

### TikTok Shop US

| 字段 | 内容 |
|---|---|
| **费率结构** | 佣金 2-8%（品类而异）；达人佣金 10-30%；投流 CPA $0.5-3；平台活动补贴 |
| **合规路径** | 与 Amazon US 相同的联邦法规；额外注意内容合规（不能虚假宣称、不能有 before/after 医疗效果）；达人内容需标注广告 |
| **竞争特征** | 内容驱动，头部集中度低于 Amazon；素材疲劳快（7-14 天）；新品可通过内容快速起量 |
| **证据优先级** | 内容播放量与转化率、达人历史表现、商品点击率、评论购买意图、素材可复制性 |
| **卖家工具数据源** | TikTok Creator Marketplace、Kalodata、FastMoss、EchoTik |
| **常见失败模式** | 只有播放无转化、素材迅速疲劳、达人佣金侵蚀利润、内容合规问题导致下架 |

### Amazon DE（德国）

| 字段 | 内容 |
|---|---|
| **费率结构** | 佣金 7-15%；FBA 费 €3-8（标准尺寸）；CPC 广告 €0.6-1.8；仓储费 €0.48-2.08/立方英尺 |
| **合规路径** | GPSR（EU 负责⼈）、CE 标志、WEEE（电子）、VerpackG（包装注册）、REACH（化学品）；DE 额外：ProdSG（产品安全法）、ElektroG（电子注册）；VAT 19% |
| **竞争特征** | 头部集中度中等；Review 壁垒中等（200-1000）；DE 消费者退货率极高（10-25%），需计入利润模型 |
| **证据优先级** | 关键词簇（德语）、首页 Review 结构、退货率基准、合规成本、VAT 计算 |
| **卖家工具数据源** | Helium 10（支持 DE）、Jungle Scout（支持 DE）、Keepa |
| **常见失败模式** | 退货率失控、合规成本低估（GPSR+EPR+多语言标签）、VAT 计算错误、德语 Listing 质量差 |

### Amazon JP（日本）

| 字段 | 内容 |
|---|---|
| **费率结构** | 佣金 8-15%；FBA 费 ¥300-800（标准尺寸）；CPC 广告 ¥50-150；仓储费月 ¥100-300/立方英尺 |
| **合规路径** | PSE（电子）、PSC（特定产品）、食品卫生法（食品/餐具）、药机法（化妆品/ quasi-drug）；JCT 10%；日语标签必须 |
| **竞争特征** | 头部集中度中等；Review 壁垒中等（100-500）；JP 消费者更看重评分质量和品牌信任，而非评论数量 |
| **证据优先级** | 关键词簇（日语）、首页评分质量、PSE/PSC 合规状态、日语 Listing 质量、包装完整性 |
| **卖家工具数据源** | Amazon JP Seller Central、Keepa（支持 JP） |
| **常见失败模式** | 包装/说明书质量不达标、日语 Listing 机翻感强、PSE 认证缺失、客服响应慢 |

### Shopee TH（泰国）

|---|---|

| **合规路径** | TISI（部分产品需泰国工业标准）、FDA（食品/化妆品/医疗）、NBC（电信设备）；进口需 Thai FDA 注册 |
| **竞争特征** | 头部集中度低；价格竞争激烈；COD 取消率 5-15%；内容电商（Shopee Live）增长快 |

| **常见失败模式** | COD 取消导致亏损、活动价格战无利润、物流时效慢影响店铺评分、跨站点照搬策略 |

### Shopify US（DTC 独立站）

| 字段 | 内容 |
|---|---|
| **费率结构** | Shopify 月费 $39-399；支付处理 2.9%+$0.30；广告 CAC $10-50（品类差异大）；建站/插件 $50-300/月 |
| **合规路径** | 与 Amazon US 相同的联邦法规；额外注意：ADA 合规（网站无障碍）、CCPA（CA 隐私）、TCPA（营销短信） |
| **竞争特征** | 无自然流量，全靠付费获客；品牌信任是核心壁垒；CAC 回收期决定生死 |
| **证据优先级** | CAC、CVR、AOV、复购率、LTV/CAC 比率、贡献毛利 |
| **卖家工具数据源** | Shopify Analytics、Google Analytics、Meta Ads Manager、Klaviyo |
| **常见失败模式** | CAC 无法回收、无复购导致 LTV 不足、落地页转化率低、信任不足（无品牌背书） |

## 组合卡片维护

1. 本文件的数值是参考区间，必须联网核验当前数据。
2. 每次评分时记录数据日期和来源。
3. 如果当前数据与本文件参考区间偏差 >30%，以当前数据为准，并在报告中注明偏差。
4. 新增高频组合时，按上述结构补充卡片。
5. 本文件每季度或平台政策重大变更时更新。

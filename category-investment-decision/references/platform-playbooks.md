# 平台分析剧本

## 使用原则

只读取用户指定或报告需要比较的平台。平台剧本改变证据优先级和运营打法，不改变七维权重。当前价格、费率、政策、物流和平台规则必须联网核验。

本文件覆盖常见平台。遇到未列明平台、平台形态混合或规则不确定时，读取 `platform-routing-universal.md`，先判断平台类型和主流量逻辑。

| 平台 | 流量本质 | 优先证据 | 常见失败模式 |
|---|---|---|---|
| Amazon | 搜索驱动的明确需求 | 关键词簇、首页结构、Review、新品突破、CPC、FBA 经济模型 | 泛词硬打、评论壁垒、广告吞噬利润、质量/退货失控 |
| TikTok Shop | 内容驱动的发现与冲动消费 | 前 3 秒、完播/互动、评论需求、达人适配、素材裂变、履约 | 只有播放无转化、素材迅速疲劳、佣金/达人费侵蚀利润 |
| Temu | 价格与供应链效率 | 核价、竞品最低价、重量体积、供货速度、退货与合规 | 核价后无利润、同质价格战、供应波动、物流属性不适配 |
| Shopify/DTC | 付费/自然获客与品牌信任 | 贡献毛利、CAC、CVR、AOV、复购/LTV、内容与 SEO | 单品毛利看似高但 CAC 无法回收、无复购、信任不足 |
| Walmart | 搜索、价格与零售可信度 | 价格竞争力、WFS/履约、品牌与库存能力、类目结构 | 低价预期压利润、库存与绩效跟不上、缺乏渠道资质 |
| Etsy | 发现、审美与个性化意图 | 原创性、工艺/定制、搜索词、视觉、礼品与季节性 | 商品不符合平台定位、同质转售、版权/商标风险 |
| eBay | 搜索、价格与稀缺/交易属性 | 成交记录、价格、商品状态、卖家信誉、跨境履约 | 标准新品缺乏优势、售后争议、价格透明导致薄利 |
| Shopee/Lazada | 站内活动、价格和本地履约 | 站点需求、活动价、广告、店铺评分、本地物流与 COD | 跨站点照搬、低价促销亏损、履约慢、COD 取消/退货 |

## Amazon

- 需求：用关键词簇、类目榜单和销量/排名代理信号交叉验证。
- 竞争：看自然位与广告位、Review 分布、评分、上架时间、品牌集中度及近 12 个月新品突破。
- 利润：使用当前佣金和 FBA 尺寸重量报价；计算盈亏平衡 ACOS。
- 切入：优先长尾人群/场景和竞品差评改良，不以换颜色或堆配件冒充差异化。
- 测试：核心观察 CTR、CVR、CPC、ACOS、自然单占比、退货与评分。

## TikTok Shop

- 需求：区分内容热度、评论购买意图和真实成交；不以播放量替代需求。
- 内容：检查前 3 秒是否理解、演示是否可重复、达人是否能自然植入、多角度素材能否持续生产。
- 利润：纳入平台费、达人佣金、样品、退款和素材成本；用 CPA/贡献利润判断放量。
- 切入：低解释成本、结果可视化和明确使用场景优先；价格带只作为站点与类目基准，不设全球硬阈值。
- 测试：观察 3 秒留存、完播、商品点击率、CVR、CPA/ROAS、退款和素材疲劳。
- 扩散：记录结果来自单条爆量、单达人、单人群，还是多素材/多达人/多地区共同有效；高播放低分散不等于可放量。

### TikTok Shop 达人视频端口

达人视频不是“寄样越多越好”，每个达人端口都要有独立经济模型。寄样前至少估算单达人固定成本、单件贡献利润、达人历史曝光能力和商品点击/成交转化。

| 模块 | 字段 | 判断方式 |
|---|---|---|
| 固定成本 | 样品、达人合作费、视频制作、基础推流 | 作为单达人端口沉没成本 |
| 可变成本 | 采购、物流、平台佣金、达人佣金、支付费、退货损耗 | 计算单件贡献利润 |
| 漏斗 | 曝光、商品点击率、点击成交 CVR | 反推最低点击和最低曝光 |
| 复用性 | 同脚本多达人复制、素材疲劳、评论购买意图 | 判断是否扩大达人池 |

进入批量寄样前运行 `scripts/reverse_funnel.py creator input.json` 或手工复核同口径公式。若达人历史平均播放低于最低曝光量，或商品点击与成交没有同类目证据，只能小样本观察，不进入批量寄样。

报告必须区分：

- 播放量高但商品点击低：内容吸引不等于购买意图。
- 商品点击高但 CVR 低：价格、页面、信任或履约承诺需要修复。
- 少数达人爆量：先验证脚本和人群是否可复制，再扩大预算。
- 反馈高度集中：先扩展相似达人/素材做复验，不直接备货或加大固定投流。
- 反馈分散但利润为负：说明传播可能成立，但商业模型不成立，先修复利润口径。

### TikTok Shop 直播端口

直播端口先看链路健康，再看 GMV。单场直播建议按用户侧、直播侧、商品侧和转化侧拆解：

| 层级 | 关键指标 | 作用 |
|---|---|---|
| 用户侧 | UV、地域、年龄、历史消费频次 | 判断流量是否匹配目标人群 |
| 直播侧 | 停留时长、互动率、转粉率、时段 | 判断内容和主播承接能力 |
| 商品侧 | 客单价、折扣力度、库存周转、合规状态 | 判断货盘和价格是否适合直播 |
| 转化侧 | 商品点击率、加购率、下单率、支付成功率 | 定位漏斗断点 |

基础公式：

`订单量 = UV × 商品点击率 × 加购率 × 下单率 × 支付成功率`

`单场利润 = 订单量 × 客单价 - 商品成本 - 推广成本 - 运营成本 - 佣金成本 - 售后/退货准备金`

直播热度不能替代利润判断。若单场利润依赖极端折扣、补贴或不可持续投流，应降级为观察或迭代，不作为放量依据。

## Temu

- 需求：核验站点价格带、动销、同款密度和生命周期。
- 供应：重点看出厂价、申报价、重量体积、交期、补货与备选工厂。
- 利润：以平台实际模式和核价规则计算，不套用 Amazon 佣金/FBA 模型。
- 切入：轻小件、成熟供应、低售后和可快速迭代优先。
- 测试：记录核价通过率、动销、履约异常、退款、实际净贡献和价格下探空间。

## Shopify/DTC

- 需求：先验证人群问题与支付意愿，再评估品牌叙事；“有故事”不能替代转化。
- 流量：拆分付费社媒、搜索、达人、邮件和自然内容的 CAC 与转化路径。
- 利润：使用贡献毛利、CAC 回收期、AOV、复购率和 LTV；不要只看产品毛利率。
- 切入：英雄品要能承载清晰价值主张，并具备加购、套装、订阅或复购路径。
- 测试：观察落地页 CTR/CVR、CAC、AOV、退款、30/60/90 天复购和回收期。

## 多平台比较

先分别按平台流量逻辑判断，再给整体投资结论。不要用同一组流量阈值横向比较不同平台；统一比较可使用净贡献、现金回收、风险和供应链复用程度。

### 平台经营逻辑差异矩阵

同一品类在不同平台上的经营逻辑完全不同。以下矩阵帮助快速识别关键差异，避免跨平台套用：

| 经营要素 | Amazon | TikTok Shop | Temu | Shopify/DTC |
|---|---|---|---|---|
| **流量获取** | 关键词搜索 + CPC 广告 + 自然排名 | 内容推荐 + 达人分发 + 直播 + 投流 | 平台分配 + 活动报名 + 价格竞争力 | 付费广告（Meta/Google）+ SEO + 邮件 + 社媒 |
| **转化驱动** | 搜索匹配度 + Review 数量/评分 + 价格 + A+ 页面 | 前 3 秒钩子 + 内容说服力 + 价格冲动 + 达人信任 | 价格 + 平台信任 + 活动标签 | 品牌信任 + 落地页质量 + 社会证明 + 运费门槛 |
| **核心壁垒** | Review 积累 + 自然排名 + 品牌注册 | 内容能力 + 达人关系 + 素材裂变效率 | 供应链效率 + 核价能力 + 周转速度 | 品牌资产 + CAC 控制 + LTV/复购 |
| **费用结构** | 佣金 8-15% + FBA 费 + CPC 广告 | 佣金 2-8% + 达人佣金 10-30% + 投流 | 平台核价分成 + 物流费 | 支付处理 2.9% + 广告 CAC + 建站费 |
| **内容要求** | Listing 文案 + A+ 图文 + 主图质量 | 短视频（前 3 秒决定生死）+ 直播 + 达人素材 | 基础图文 + 活动素材 | 品牌视觉 + 落地页 + 邮件 + 社媒内容矩阵 |
| **供应链要求** | FBA 入仓标准 + 包装规范 + 库存管理 | 轻小件优先 + 发货时效 + 退货处理 | 极致成本控制 + 快速补货 + 重量体积敏感 | 灵活发货 + 品牌包装 + 质检一致性 |
| **利润模型** | 售价 - 产品成本 - FBA - 佣金 - CPC = 净利 | 售价 - 产品成本 - 平台费 - 达人佣金 - 投流 = 净利 | 核价 - 产品成本 - 物流 = 薄利 | 售价 - 产品成本 - CAC - 运营 = 净利（靠 LTV 回收） |
| **对选品的含义** | 需要搜索需求验证 + Review 可攻破性 + 广告可承受 | 需要内容可演示性 + 达人可分发 + 冲动转化力 | 需要极致成本 + 轻小件 + 快速周转 | 需要品牌故事 + 高毛利 + 复购/LTV 路径 |

### 平台切换检查清单

当同一品类需要同时评估多个平台，或从已有平台扩展到 new 平台时，逐项检查以下差异：

1. **流量逻辑切换**：新平台的主要流量来源是什么？是搜索驱动还是内容驱动？我的内容能力是否匹配？
2. **费用结构切换**：新平台的费用结构是什么？佣金、广告、物流、达人佣金的占比如何？利润模型是否成立？
3. **证据优先级切换**：在新平台上，哪些证据最重要？（Amazon 看 Review 和关键词，TikTok 看内容和达人，Temu 看核价和周转）
4. **竞争结构切换**：新平台的竞争格局是什么？头部集中度、新品突破难度、广告压力如何？
5. **运营负荷切换**：新平台需要什么样的运营能力？我的团队是否具备？需要额外投入什么？
6. **供应链切换**：新平台对供应链有什么特殊要求？（FBA 入仓 vs 自发 vs 平台物流）
7. **时间节奏切换**：新平台的放量节奏是什么？（Amazon 8-12 周 vs TikTok 4-8 周 vs Temu 2-4 周）

**禁止跨平台套用的硬规则**：

- 不得把 Amazon 的 Review/CPC 逻辑套用到内容电商（TikTok、Instagram）。
- 不得把 TikTok 的播放量等同于搜索电商的需求信号。
- 不得把 Temu 的核价逻辑套用到 DTC（独立站靠品牌溢价，不靠最低价）。
- 不得把独立站的毛利空间等同于 CAC 可回收（必须验证 CAC 回收期）。
- 不得把"能在某平台上架"等同于"合规可卖"（平台准入 ≠ 法规合规）。

### 多平台组合策略

当同一品类同时进入多个平台时，注意：

- **流量协同 vs 蚕食**：多平台是带来增量用户，还是同一批用户在不同平台比价？
- **内容复用度**：一个平台的内容素材能否复用到其他平台？（Amazon Listing 图文 → DTC 落地页；TikTok 短视频 → Instagram Reels）
- **供应链复用度**：同一产品能否满足多个平台的包装、质检、入仓要求？
- **运营负荷叠加**：多平台运营的团队负荷是否超出承载力？是否需要分阶段进入？
- **风险分散 vs 集中**：多平台是否真正分散了风险，还是把所有平台的费用和风险都叠加了？

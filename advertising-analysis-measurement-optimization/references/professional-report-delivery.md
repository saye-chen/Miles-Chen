# 广告专业报告与交付协议

## 报告层级

| 类型 | 使用场景 | 不可省略 |
|---|---|---|
| Advertising Decision Card | 单一异常、可行性、局部 Scale/Stop | 对象四轴、门禁、结论、关键证据、经济影响、动作、停止/回滚 |
| Advertising Decision Memo | 单平台账户、活动、测量或预算问题 | 决策页、三本账、九层诊断、计算、实验/补证、动作表、Ledger |
| Advertising Diligence | 多平台/国家、大预算、复杂归因、完整系统 | 全部章节、平台附录、敏感性、跨域冲突、审批和自检 |
| Measurement Audit | 追踪、归因、实验、MMM、品牌测量 | 事件/身份/Consent、数据流、归因限制、反事实、偏差、修复与复测 |
| Scale/Stop Review | 放量、降量、暂停、重建、停投复盘 | 成熟利润、平均/边际、容量、阶段、情景、停止和回滚 |

## 决策页

正式报告第一屏必须回答：对象/国家/平台/时窗/四轴/生命周期；`Scale / Hold / Repair / Reduce / Stop / Inconclusive`；一句话理由及为何不是相邻档；数据、追踪、成熟和增量等级；平台归因账、成熟经营账、因果增量账；当前预算与建议变化；最危险约束；3项动作；成功、停止和回滚；需要哪个主权域审批。

低置信度不得输出 Scale；追踪、费用承担、成熟利润或不可补偿红线失败时，只能 Repair/Stop/Inconclusive。广告动作不能自动改变品类投资、库存、Listing、内容机制、品牌或客户触达结论。

## 完整报告骨架

1. **封面与版本**：任务、对象、国家/平台、`D09-2026.07`、证据截止日、币种/税口径。
2. **广告决策页**：姿态、结论等级、预算变化、三本账、约束和审批。
3. **任务、对象与四轴**：最小诊断单元、流量场景、控制模式、计费、优化目标、生命周期。
4. **输入与数据血缘**：账户/活动/商品/素材/查询/人群/页面/订单数据，来源、窗口、时区、币种、延迟、缺失。
5. **追踪与测量审计**：事件、去重、Consent、身份、转化动作、价值、归因窗、订单成熟和真实曝光。
6. **Hard gates**：资格/审核/可售、利润/现金、库存履约服务、授权合规和平台事故。
7. **影响范围与九层诊断**：范围、根因树、支持、反证、替代解释、待验证变量。
8. **平台专项**：按对应平台专家卡写特有对象、字段、限制和禁止推断；动态事实附当前来源。
9. **广告经济**：完整成本、保本/目标、成熟贡献利润、平均/边际 ROAS/CPA/CPC、敏感性和安全空间。
10. **归因与增量**：平台归因可用范围、自然/品牌/再营销蚕食、实验或准实验、iROAS和外推边界。
11. **预算与出价方案**：探索/验证/放量/防守分池，动作对象、幅度、阶梯、上限和依赖。
12. **实验与观察计划**：假设、随机/分析单位、主要/护栏/长期指标、MDE/样本、污染和停止。
13. **跨域冲突与路由**：D01/D02/D06-D13输入、冲突、状态、允许/禁止用途和待审批项。
14. **执行与回滚**：负责人角色、先后顺序、观察窗、成功、停止、回滚和复核日期。
15. **Evidence / Assumption / Calculation / Decision Ledger**：关键结论逐项回指。
16. **自检摘要**：未核验事实、降级项、失败工具、P0/P1、模型版本和文件校验。

## 核心表格

| 表 | 必备列 |
|---|---|
| 对象四轴表 | 对象ID、国家/平台、场景、控制、计费、目标、阶段、窗口 |
| 数据与测量表 | 字段/事件、来源、时区/币种、完整性、延迟、成熟、允许用途、缺口 |
| 九层诊断表 | 层、异常、基线、范围、根因、支持、反证、替代解释、动作 |
| 三本账表 | 平台归因、成熟经营、因果增量、口径、窗口、区间、禁止用途 |
| 经济与边际表 | 档位、花费、成熟收入、贡献利润、平均效率、边际效率、安全空间 |
| 行动表 | 动作对象、当前值、目标/幅度、预算、责任人、观察窗、成功、停止、回滚 |
| 跨域裁决表 | 来源Skill、对象/版本、建议、状态、主权域、接受/拒绝原因、影响 |

## Ledger 最小字段

- Evidence：ID、类型、来源、对象、国家/平台、观察/访问时间、指纹、质量、支持/反对、限制。
- Assumption：ID、内容、取值/范围、敏感性、推翻条件、补证动作。
- Calculation：ID、脚本/版本、输入哈希、币税口径、结果、区间、复算状态。
- Decision：ID、主权域、结论、所依赖 E/A/C、状态、允许/禁止用途、审批和复核日。

## 文件交付与质量闸门

报告不是聊天拼接。完整报告必须通过仓库级报告评分器：总分至少85，且证据、经济、主权、动作/停止四个关键维度分别不少于该维度80%；出现伪造事实、主权越界、利润不可复算、因果越界、未解决红线却Scale任一项直接失败。
